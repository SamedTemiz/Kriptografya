# 🔐 CRYPTOGRAPHY WEB GAME - DETAYLI PROJE PLANI

## 📋 PROJE ÖZETİ
- **Proje Adı**: Cryptography Web Game
- **Teknoloji**: Next.js 14 + TypeScript + Tailwind CSS
- **Hedef**: Eğitici ve eğlenceli kriptografi oyunu
- **Platform**: Web (Responsive Design)
- **Monetization**: Freemium model (Ads + Premium)

## 🎯 OYUN KONSEPTİ
- **Ana Mekanik**: Türkçe cümlelerin harflerini sayılarla gizleme
- **Hedef**: Gizlenen cümleyi 3 hata hakkıyla bulma
- **Zorluk**: Aynı harfler aynı sayıya karşılık gelir (Substitution Cipher)
- **Eğitim**: Türkçe kelime dağarcığı ve mantıksal düşünme

## 🎮 OYUN ÖZELLİKLERİ

### **Temel Oyun Mekaniği**
```
🎯 Game Flow:
1. Kullanıcı zorluk seviyesi seçer
2. Sistem rastgele cümle seçer
3. Cümle harfleri sayılarla gizlenir
4. Kullanıcı harf-sayı eşleştirmesi yapar
5. 3 hata hakkı ile cümleyi bulmaya çalışır
6. Skor hesaplanır ve kaydedilir
```

### **Zorluk Seviyeleri**
```
📊 Difficulty Levels:
├── 🟢 Kolay
│   ├── 3-4 kelimeli Türkçe cümleler
│   ├── Günlük yaşam kelimeleri
│   ├── Basit gramer yapıları
│   └── 5 dakika süre
├── 🟡 Orta
│   ├── 5-7 kelimeli Türkçe cümleler
│   ├── Çeşitli konular
│   ├── Orta seviye kelimeler
│   └── 7 dakika süre
└── 🔴 Zor
    ├── 8+ kelimeli Türkçe cümleler
    ├── Akademik/edebi terimler
    ├── Karmaşık gramer yapıları
    └── 10 dakika süre
```

### **Türkçe Karakter Desteği**
```
🔤 Character Support:
├── Türkçe alfabe (29 harf)
├── Özel karakterler (ç, ğ, ı, ö, ş, ü)
├── Büyük/küçük harf desteği
├── Noktalama işaretleri
├── Sayılar ve semboller
└── Unicode encoding (UTF-8)
```
